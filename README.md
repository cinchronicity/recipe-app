# Recipe App

This is a Django-based Recipe application that allows users to create and manage recipes with ingredients, cooking time, and descriptions. The application uses Django models and the Django admin panel to store and manage recipe data, featuring a robust database structure with relationships between recipes and ingredients. This project was built as part of a learning exercise to practice database modeling, relationships, and testing in Django while developing practical web application skills.

## Features

- Recipe creation and management
- Ingredient tracking and organization
- Django admin interface for data management
- SQLite database integration
- Unit testing capabilities

## Technologies Used

- Python
- Django
- SQLite
- HTML/CSS (Django templates)
