# Default Recipe Image

For your actual implementation, you'll want to add a real image file here.

You can:

1. Download a free placeholder image (300x200px recommended)
2. Save it as 'default-recipe.jpg' or 'default-recipe.png'
3. Place it in: static/images/
4. Reference it in templates with: {% static 'images/default-recipe.jpg' %}

Good sources for placeholder images:

- Unsplash (unsplash.com)
- Pixabay (pixabay.com)
- Or create a simple colored rectangle with text "Recipe Image"
